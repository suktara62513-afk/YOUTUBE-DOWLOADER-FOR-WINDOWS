# 🚀 Youtube Downloader for Windows

> A modern, beautiful and powerful Terminal-based YouTube Downloader for Windows.

Built with **PowerShell**, powered by **yt-dlp** and **FFmpeg**.

Made with ❤️ by **Tarek X Antigravity**

---

# ✨ Features

- 🎨 Modern Sci-Fi Terminal UI
- ⚡ Fast YouTube Downloader
- 🎬 Download videos in the highest available quality
- 🎵 Download audio as MP3
- 📺 Easy quality selection menu
- 📊 Dynamic progress bar
- 🔄 Automatic video & audio merge
- 🤖 Auto dependency detection
- 📥 Automatically downloads required tools
- 💻 Native Windows PowerShell application
- 🆓 Free & Open Source

---

# 📂 Project Structure

```
Youtube Downloader
│
├── Downloader.ps1     # Main application
├── run.bat            # Double-click to start
├── build.bat          # Optional build/helper script
└── README.md
```

---

# 📋 Requirements

- Windows 10 or later
- Internet connection
- PowerShell (built into Windows)

No manual installation of **yt-dlp** or **FFmpeg** is required.

The downloader will automatically download and configure missing dependencies on first launch.

---

# 🚀 Getting Started

## Step 1

Download or clone this repository.

```
git clone YOUR_GITHUB_REPOSITORY
```

or simply download the ZIP file.

---

## Step 2

Extract the project.

---

## Step 3

Double-click

```
run.bat
```

That's it!

The downloader will:

- Check dependencies
- Download missing tools automatically
- Launch the downloader
